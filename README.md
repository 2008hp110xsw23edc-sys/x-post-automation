# X投稿 自動生成システム

毎日5:00（JST）にClaude APIがX投稿案を自動生成し、`posts/` フォルダに保存します。

## フロー

```
毎日5:00 → Claude APIが生成 → posts/YYYY-MM-DD.md に保存 → GitHubで確認・修正 → Xに投稿
```

## 生成内容

毎日以下の6セットを生成します：

| タイプ | 内容 | セット数 |
|---|---|---|
| 共感型 | 読者の悩みや感情を言語化 | 2 |
| ストーリー型 | 自分の経験・進捗・葛藤をリアルに語る | 2 |
| 問いかけ型 | 読者に「自分ごと」として考えさせる | 2 |

## 投稿の使い方

1. `posts/YYYY-MM-DD.md` をGitHubで開く
2. 内容を確認・修正する
3. 気に入った投稿をXにコピー＆ペーストして投稿
